## ğŸ” Technical Details

### Calculation Formula

```javascript
const numberOfDays = Math.floor(amount / dailyTarget);
const remainder = amount % dailyTarget;
const totalDays = numberOfDays + (remainder > 0 ? 1 : 0);
```

### Distribution Logic

```javascript
// For each full day
for (let i = 0; i < numberOfDays; i++) {
  const date = startDate + i days;
  contribute(date, dailyTarget);
}

// For remainder day (if exists)
if (remainder > 0) {
  const finalDate = startDate + numberOfDays days;
  contribute(finalDate, remainder);
}
```

### Transaction Recording

**Single Transaction:**
- ID: Timestamp

- Amount: Full lump sum
- Date: Starting date
