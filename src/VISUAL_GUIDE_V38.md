## Responsive Behavior

### Small Screen (< 640px):
- Stacked layout
- Full-width cards
- Bottom navigation
- Single column

### Medium Screen (640px - 1024px):
- 2-column grid where appropriate
- Expanded cards
- More white space

### Large Screen (> 1024px):
- Sidebar navigation (admin)
- 3-column grids
- Maximum width container
- Desktop-optimized

## Animation & Transitions

### Toast Notifications:
```
Entrance: Slide in from top (0.3s)
Exit: Fade out (0.2s)
Hover: Slight shadow increase
```

### Reminder Cards:
```
Appear: Fade in + slide down (0.4s)
Dismiss: Fade out + slide up (0.3s)
Hover: Scale 1.02
```

### Button States:
```
Hover: Background darken
Active: Scale 0.98
Loading: Spinner animation
```
