## Migration Guide

### For Existing Users

**Email-Only Accounts:**
```
Status: Still work
Login: Use email
Action: Add phone in profile (optional)
Verification: Already verified
```

**New Signups:**
```
Status: Phone required
Login: Use phone or email
Action: Verify phone during signup
Verification: SMS code
```

### Database Update

**Old User Format:**
```json
{
  "email": "user@example.com",
  "phone": "08012345678",
  "emailVerified": true
}
```

**New User Format:**
```json
{
  "email": "user@example.com",
  "phoneNumber": "+2348012345678",
  "phoneVerified": true
}
```

**Migration Script:**
```javascript
const users = JSON.parse(localStorage.getItem("users") || "[]");
users.forEach(user => {
  // Normalize phone
  if (user.phone && !user.phoneNumber) {
    user.phoneNumber = normalizePhoneNumber(user.phone);
  }
  // Set verification status
  if (user.emailVerified && !user.phoneVerified) {
    user.phoneVerified = true;
  }
});
localStorage.setItem("users", JSON.stringify(users));
```
